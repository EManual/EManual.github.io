jsp中嵌入java代码的方式：
1) 表达式标签 <%=  1 + 1 %> 
a) 计算表达式的返回值。
b) 能将返回值在网页上显示出来。
不能出现 ";" 
“ <%= 1+1%>”'则在网页上显示2
2) 声明标签：<%!    %> 
用来声明变量和函数，在声明标签中声明的变量和函数，可以在本页面的其他的java代码中使用。声明的位置是首是尾皆无妨。建议尽量少声明变量，因为jsp最终要被解释为servlet，声明的变量在servlet中就体现为实例变量，在多个线程访问这个servlet的时候, 由于实例变量被多个线程共享使用(实例变量线程不安全，局部变量线程安全)，有可能出现问题，并且不好解决。
3) 普通脚本：“<%   %>”'  普通脚本是不能嵌套的。
```java  
	<%
	  for( int i = 0 ; i < 10 ; i++ ){
		<%= 1+ 1 %>
	  }
	%>
```
